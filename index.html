CSS Cheatsheet

Typography

Property Description Values
font-family allows you to change font of HTML elements 'Times New Roman', 'Arial', 'serif', 'cursive'
font-weight allows you to bold/thin font normal - 400, bold - 700, heavy bold - 900
text-transforms allows you to style text all uppercase, lowercase and more uppercase, lowercase, intial, capitalize

Text Layout

Property Description Values
letter-spacing used to space individual letters in an element, can make fonts more readable 2px
word-spacing spaces words in an element, can make large/bolded text more readable 0.3em
line-height sets the height of a line containing text, numerical values will auto-adjust if you change font size 1.4
text-align aligns the text in an element right, center, left, justify

Colors

Color Type Description Values
Named Colors english words for colors blue, pink, yellow
RGB numeric values that describe a mix of red, green and blue #FFFFFF or rgb (23, 45, 23)
HSL numeric values that describe the mix of hue, saturation and light hsl (120, 60%, 70%
Alpha numeric value that adjusts the level of opacity 0 (transparent) 1 (opaque)
